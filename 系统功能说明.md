# 智能家居销量数据分析系统 - 功能说明

## 系统功能概述

本系统包含以下核心功能：

### 1. 用户认证功能
- **登录**: 用户可以使用用户名和密码登录系统
- **注册**: 新用户可以注册账号
- **路由守卫**: 未登录用户无法访问需要认证的页面

### 2. 数据管理功能
数据管理功能用于管理清洗后上传到MySQL的数据，支持以下6个数据表的CRUD操作：
- **销售汇总表** (sales_summary)
- **地域分析表** (region_analysis)
- **产品分析表** (product_analysis)
- **用户行为表** (user_behavior)
- **预测结果表** (prediction_result)
- **特征重要性表** (feature_importance)

每个表都支持：
- 查看所有数据
- 新增数据
- 编辑数据
- 删除数据

### 3. 可视化列表
可视化列表页面展示6个可视化分析模块，每个模块对应一个独立的可视化图表：

1. **销售趋势分析** - 展示销售额随时间的变化趋势
2. **地域分布分析** - 展示不同地区的销售分布情况
3. **产品品类分析** - 展示不同产品品类的销售占比
4. **用户画像分析** - 展示用户性别、会员类型等画像信息
5. **销售渠道分析** - 展示不同销售渠道的销售情况
6. **随机森林预测** - 展示随机森林模型的预测结果和特征重要性

### 4. 数据分析流程

#### Python部分（三步独立操作）

**步骤1: 数据清洗**
- 文件: `python_analysis/01 数据清洗.py`
- 功能: 读取Excel文件，清洗数据，合并数据，去除重复列，保存为单个CSV文件
- 输出: `output/cleaned_data/merged_data.csv`

**步骤2: 数据分析（从MySQL读取）**
- 文件: `python_analysis/02_数据分析_从MySQL.py`
- 功能: 从MySQL读取清洗后的数据，执行多种数据分析，保存分析结果为多个CSV文件
- 输入: MySQL数据库中的清洗后数据
- 输出: `output/analysis_results/` 目录下的多个CSV文件

**步骤3: 数据存储**
- 文件: `python_analysis/03 数据合并.py`
- 功能: 从本地CSV文件读取所有分析结果，保存到MySQL数据库
- 输出: MySQL数据库中的分析结果表

#### 数据分析基于MySQL数据
- 数据分析模块 (`data_analysis_from_mysql.py`) 从MySQL读取清洗后的数据
- 如果MySQL中没有数据，会自动回退到从CSV文件读取
- 分析结果保存为CSV文件，然后通过步骤3上传到MySQL

#### 随机森林预测
- 使用随机森林回归模型进行销售额预测
- 分析特征重要性
- 预测结果和特征重要性都会保存到MySQL数据库

## 系统架构

### 前端 (Vue 3 + Element Plus)
- **登录页面**: `/login`
- **Dashboard**: `/dashboard` - 展示所有可视化图表
- **可视化列表**: `/visualization` - 6个可视化模块入口
- **数据管理**: `/data-management` - 数据CRUD操作

### 后端 (SpringBoot 2.7.x)
- **认证API**: `/api/auth/login`, `/api/auth/register`
- **数据管理API**: `/api/data/manage/*` - 支持所有表的CRUD
- **分析结果API**: `/api/analysis/*` - 获取分析结果数据
- **预测API**: `/api/prediction/*` - 获取预测结果和特征重要性

### 数据库 (MySQL 5.6.x)
- **用户表**: `sys_user`
- **分析结果表**: `sales_summary`, `region_analysis`, `product_analysis`, `user_behavior`, `prediction_result`, `feature_importance`

## 使用流程

### 1. 初始化数据库
执行 `Smart_Home_Analysis/src/main/resources/db/schema.sql` 和 `user_table.sql` 创建数据库表结构

### 2. 运行Python分析
```bash
cd python_analysis
# 步骤1: 数据清洗
python "01 数据清洗.py"

# 步骤2: 数据分析（从MySQL读取）
python "02_数据分析_从MySQL.py"

# 步骤3: 数据存储
python "03 数据合并.py"
```

### 3. 启动后端
```bash
cd Smart_Home_Analysis
mvn spring-boot:run
```

### 4. 启动前端
```bash
cd Smart_Home_Analysis/frontend
npm run serve
```

### 5. 访问系统
- 前端地址: http://localhost:8081
- 后端API: http://localhost:8080

## 注意事项

1. **MySQL配置**: 确保Python脚本中的MySQL连接配置正确
2. **数据流程**: 数据清洗 -> 数据存储到MySQL -> 数据分析从MySQL读取 -> 分析结果保存到MySQL
3. **用户认证**: 首次使用需要注册账号，然后登录
4. **数据管理**: 数据管理功能用于管理MySQL中的分析结果数据，支持增删改查
